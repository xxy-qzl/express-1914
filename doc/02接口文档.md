# 接口文档

## 前后端交互通信的桥梁

## 文章新增

1. 请求地址：http://localhost:8080/api/posts

2. 请求方式：POST

3. 请求参数：json

| 参数名称 | 参数类型 | 是否必须 | 参数描述 |
| ------- | ------- | ------- | -------  |
|  title  |  String  |  Y     | 文章标题  |
|  body   |  String  |  Y     | 文章正文  |

4. 响应结果

```json
{
    "code":0,
    "msg":"OK",
    "data":{
        "id":"xxx",
        "title":"xxx",
        "body":"xxx"
    }
}
```
